■이 압축 파일에 대해
  이 압축 파일은 《프로가 되기 위한 웹 기술 입문》에서 소개한 샘플 소스 코드를 배포한
  것입니다.



■이 압축 파일의 구성
  이 압축 파일은 다음과 같은 폴더로 구성되어 있습니다. 
  
  ├ mock              Lesson4 피자 펜토미노 화면 모형
  ├ pentomino_1       Lesson4 피자 펜토미노(PHP) 인증 기능만
  │   (http://www.mobills.co.kr/webtext/pentomino_1/login.html에서 확인 가능)
  │
  ├ pentomino_2       Lesson4 피자 펜토미노(PHP) Cookie사용판
  │   (http://www.mobills.co.kr/webtext/pentomino_2/login.html 에서 확인 가능)
  │
  ├ pentomino_3       Lesson4 피자 펜토미노(PHP) 완성판
  │   (http://www.mobills.co.kr/webtext/pentomino_3/login.html 에서 확인 가능)
  │
  ├ PizzaPentomino    Lesson6 피자 펜토미노(Java) Struts판
  │   (http://www.mobills.co.kr/webtext/PizzaPentomino/  에서 확인 가능)
  │
  └ PizzaPentominoSrv Lesson6 피자 펜토미노(Java) 서블릿판
  
  ※ Java판은 Eclipse에서 가져오기가 가능한 형식으로 되어 있습니다
